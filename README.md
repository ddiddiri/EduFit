# EduFit 프로젝트

## 프로젝트 개요

**EduFit**은 대한민국 교육 현장의 AI·코딩 교육 문제를 해결하기 위한 학교 맞춤형 교육 설계 플랫폼입니다.

### 프로젝트 배경

근 10년간 대한민국 교육계에 코딩 열풍이 불면서 많은 학교에 코딩 교육 자원(장비, 예산, 인프라)이 확충되었습니다. 하지만 실제 교육 현장에서는 다음과 같은 문제들이 발생하고 있습니다:

- **교육 목표의 불확실성**: 무엇을 위한 코딩 교육인지 명확하지 않음
- **교사의 전문성 부족**: 비전공 교사가 코딩을 가르치는 경우가 많음
- **학교 자원 활용률 저조**: 장비는 있지만 제대로 활용하지 못함

**결과적으로 "자원은 있는데 제대로 된 교육은 받지 못하는" 상황이 발생**하고 있으며, EduFit은 이 문제를 해결하고자 합니다.

## 프로젝트 목표

### 주요 목표
- **포트폴리오**: 실제 동작하는 풀스택 프로젝트 완성
- **사회 공헌**: 교육 현장의 실질적 문제 해결

### 서비스 목표
1. **학교별 맞춤형 AI·코딩 교육 계획 제공**
2. **교사의 업무 부담 감소 및 전문성 향상 지원**
3. **보유 자원의 효율적 활용 방안 제시**

## 타겟 사용자

### 주 타겟: 정보/컴퓨터 담당 교사
- **연령대**: 30-50대
- **특징**: 실제 수업 진행, 비전공자 多
- **니즈**: 
  - 바로 쓸 수 있는 커리큘럼
  - 학교 환경에 맞는 교육 계획
  - 교사 역량 강화

### 부 타겟: 학교 관리자 (교감/교장)
- **연령대**: 40-60대
- **특징**: 예산 및 의사결정권자
- **니즈**: 
  - 명확한 교육 성과
  - 투자 대비 효과
  - 학교 평판 개선

## 주요 기능

### 1. 학교 정보 수집 (멀티 스텝 폼)

**5단계로 구성된 체계적인 정보 수집**

- **Step 1: 기본 정보**
  - 학교명, 지역, 학교 유형, 학생 수, 학급 수
  - NEIS API를 통한 학교 검색 자동완성

- **Step 2: 운영 현황**
  - 현재 코딩 교육 진행 여부
  - 사용 중인 교구/프로그램
  - 주당 수업 시간

- **Step 3: 보유 자원**
  - 컴퓨터실, AI 실습실, 로봇 교구 등
  - 학교가 보유한 시설 및 장비

- **Step 4: 교육 목표 & 애로사항**
  - 우선 교육 목표 (단일 선택)
  - 현재 겪는 어려움 (최대 3개 선택)

- **Step 5: 연락처**
  - 담당 교사 정보
  - 개인정보 수집 동의

**폼 기능 특징:**
- 진행률 표시
- 이전 단계로 돌아가기 가능
- 스텝별 검증
- React Hook Form을 통한 폼 상태 관리

### 2. 맞춤형 교육 계획서 생성

**수집된 데이터를 기반으로 학교별 맞춤 계획서 작성**

**계획서 구성:**
1. 학교 현황 분석
2. 강점 및 활용 가능 자원
3. 추정 니즈 및 목표
4. 제안 교육 프로그램
   - 프로그램명
   - 대상 학년
   - 교육 내용
   - 기간 및 시간
   - 기대 효과
5. 실행 로드맵 (단계별)
6. 예상 비용
7. 다음 단계 안내

**생성 방식:**
- 담당자가 템플릿 기반으로 작성
- 학교 데이터 자동 입력

### 3. 관리자 기능(개발 예정)

**신청 내역 관리 및 계획서 작성**

- 대시보드 (통계 요약)
- 신청 목록 관리
- 신청 상세 정보 확인
- 계획서 작성 및 발송
- 진행 상황 관리

## 기술 스택

### Frontend
- **React Native** (크로스 플랫폼 모바일 앱)
- **React** (웹)
- **TypeScript**
- **React Hook Form** (폼 관리)
- **Zod** (스키마 검증)
- **Tailwind CSS** (웹 스타일링)
- **NativeWind** (React Native 스타일링)

### Backend
- **Supabase** (BaaS)
  - Database (PostgreSQL)
  - Storage
  - Real-time subscriptions

### 외부 API
- **NEIS Open API** (학교 정보 검색)
